# 기상 데이터 수집 프로젝트 완료 요약

## 프로젝트 개요

한국 기상청 공공데이터 API를 활용하여 기상 예보, 일출일몰, 조력 정보를 수집하는 Python 프로그램입니다.

---

## ✅ 완료된 기능

### 1. 기상 예보 데이터 수집
- ✅ 24시간 예보 데이터 (온도, 습도, 날씨, 풍속, 풍향, 파고)
- ✅ 강수량, 강수확률, 적설량
- ✅ 최고/최저 기온
- ✅ 초단기예보 및 단기예보 지원
- ✅ 위경도 → 격자 좌표 자동 변환

### 2. 일출일몰 정보 수집
- ✅ 공공데이터포털 API 연동 (XML 파싱)
- ✅ 일출, 일몰, 월출, 월몰 시간
- ✅ 일중, 월중 시간
- ✅ 시민박명, 항해박명, 천문박명
- ✅ API 실패 시 천문학적 계산 방식 자동 대체

### 3. 조력(조수) 정보 수집
- ✅ 조석 예보(시계열) API 연동 (기본 사용, 미래 예측 데이터 제공)
- ✅ 현재 시간 기준 +24시간 미래 예측 데이터 수집
- ✅ 예측조위 (조위높이 cm 단위)
- ✅ 예보지점 정보 (이름, 위도, 경도)
- ✅ 예측일시 정보
- ✅ 시간 간격 조절 가능 (1분 ~ 60분)
- ✅ 여러 관측소 지원 (인천, 부산 등)

### 4. 데이터 관리
- ✅ JSON 형식으로 데이터 저장
- ✅ 과거 데이터 조회 기능
- ✅ 여러 지역 동시 조회

---

## 사용된 API

### 1. 기상청 단기예보조회서비스
- **초단기예보 URL**: `http://apis.data.go.kr/1360000/VilageFcstInfoService_2.0/getUltraSrtFcst`
- **단기예보 URL**: `http://apis.data.go.kr/1360000/VilageFcstInfoService_2.0/getVilageFcst`
- **문서**: https://www.data.go.kr/data/15084084/openapi.do
- **상태**: ✅ 정상 작동
- **특징**: 단기예보 3시간 간격 데이터를 1시간 단위로 선형 보간하여 24시간 데이터 제공

### 2. 일출일몰정보 조회서비스
- **URL**: `http://apis.data.go.kr/B090041/openapi/service/RiseSetInfoService/getAreaRiseSetInfo`
- **문서**: https://www.data.go.kr/data/15012688/openapi.do
- **상태**: ✅ 정상 작동

### 3. 조석(조력) 예측 정보
- **조석 예보(시계열) API** (기본 사용): ✅ 정상 작동
  - **URL**: `http://apis.data.go.kr/1192136/tideFcstTime/GetTideFcstTimeApiService`
  - **문서**: https://www.data.go.kr/data/15156022/openapi.do
  - **특징**: 현재 시간 기준 +24시간 미래 예측 데이터 제공
- **조석 예측 정보 API** (과거 데이터용)
  - **URL**: `http://apis.data.go.kr/1192136/surveyTideLevel/GetSurveyTideLevelApiService`
  - **문서**: https://www.data.go.kr/data/15142507/openapi.do

## 테스트 결과

### 일출일몰 API
- ✅ 서울 지역 데이터 수집 성공
- ✅ XML 응답 파싱 정상 작동
- ✅ 모든 일출몰 관련 정보 수집 성공

### 조력 API
- ✅ 인천 관측소 (DT_0001): 100개 데이터 수집 성공
- ✅ 부산 관측소 (DT_0004): 100개 데이터 수집 성공
- ✅ JSON 응답 파싱 정상 작동
- ✅ 예측조위 및 실측조위 수집 성공

### 통합 테스트
- ✅ 기상 예보 + 일출일몰 + 조력 정보 통합 수집 성공
- ✅ 여러 지역 동시 조회 정상 작동

## 주요 클래스

### WeatherAPI
- 기상청 단기예보 API 연동
- 위경도 → 격자 좌표 변환
- 24시간 예보 데이터 수집

### SunMoonAPI
- 일출일몰 정보 API 연동
- XML 응답 파싱
- 천문학적 계산 방식 (대체)

### TideAPI
- 조석 예측 정보 API 연동
- JSON 응답 파싱
- 여러 관측소 지원

### WeatherDataCollector
- 통합 데이터 수집
- 여러 지역 동시 조회
- 데이터 저장 및 관리
