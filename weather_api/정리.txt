수요예측 시스템 
○ 해상 교량 기본 정보
  - 위치(주소, 위도, 경도)
  - 조명(가로등)
  - 발전기 정보
    - 태양광, 풍력, 조력 등
    - 설비 스펙(태양광 면적, 풍력 터빈 정보 등)
  - ESS 정보
    - 전력 용량
    - 충방전 스펙 등

○ 재생에너지 발전량 예측
  - 지역 기상데이터 활용(예보)
    - 각 지역별 기상정보 수집 (시간별)
      - 온도, 습도, 날씨(맑음, 구름, 비 등)
      - 일출, 일몰 시간
      - 바람 정보(풍속, 풍향)
      - 조력 정보
  - 태양광 발전량 예측
    - 태양광 발전 기본 정보
      - 설치 면적 or 패널 단위 스펙 정보
      - 패널 개수
    - 기상 데이터
      - 과거 실측 데이터(설치 기기 센서)
      - 미래 예측 데이터(기상청 예보 데이터)
  - 풍력 발전량 예측
    - 풍력발전 기본 정보 연동
      - 블레이드 사이즈, 터빈, 등
    - 기상 데이터
      - 과거 실측 데이터(설치 기기 센서)
      - 미래 예측 데이터(기상청 예보 데이터)
  - 조력 발전량 예측
    - 조력발전 기본정보 연동
      - 블레이드 스펙 등
    - 조력 정보 없을 경우 Default 값 처리 방안
      - 각 지역별 조력정보 일괄 적용 고려
      - 과거 데이터만으로 예측 진행(발전량, 센서 정보 등)
  - 각 상태 정보는 모두 입력값으로 사용자 선택 가능
  - 각 발전량은 시간별 예측 필요

○ ESS 충방전 알고리즘
  - 기본 스펙(충방전 등)
  - 150Kw 스펙
  - 배터리 잔량에 대해 재생에너지or한전으로 충전 결정
  - 경부하(충전), 중부하(방전) 때 단가가 다르기에 충방전 계획 필요
  - 빙결, 부식 이외에 조명 등 전력 사용

○ 교량 에너지 소비량 예측
  - 동계 : 도로결빙방지설비 에너지 소비량 예측
    - 기상정보를 반영한 예측 필요
    - 겨울철 상시에 결빙 위험(조건 확인 필요)
  - 사계 : 교량 부식방지설비 에너지 소비량 예측
    - 기상정보가 예측값에 영향을 미치는지 확인 필요
      - 바닷바람?

○ 시뮬레이션 기능(화면 설계)
  - 설정 페이지(탭1)
    - 시간 설정
      - 예측 범위(6시가/12시간/24시간)
    - 기상 설정
      - 외부환경 변경 적용
      - 지역, 온도/습도/풍속/풍향
        - 기상청 API 정보 or 사용자 직접 설정
    - 발전기 설정
      - 태양광
        - 패널 스펙(모델 선택)
          - 또는 수동으로 스펙 입력
        - 패널 수량
        - 설치 각도, 방위각 설정 (필요 시)
      - 풍력
        - 터빈 스펙(모델 선택)
          - 또는 수동으로 스펙 입력(블레이드 길이(m), 허브 높이(m))
        - 풍속 컷인, 컷아웃, 정격풍속 입력
      - 조력
        - 터빈 스펙(모델 선택)
          - 또는 수동으로 스펙 입력
    - 소비 설정
      - 소비원 선택
        - 부식방지, 빙결방지
        - 조명(가로등) 등 기타 소비원
      - 소비원 스펙 설정
        - 부식방지 시스템
          - 적용 구간 길이 (m)
          - 소비 전력 (W/m)
        - 빙결방지 시스템
          - 작동 조건 설정: 예. 온도 0도 이하 + 습도 70% 이상
        - 조명 및 기타
          - 조명 소비전력, 수량, 시간 설정 (예: 19:00~06:00)
      - 전력 사용 조건 설정(빙결방지 n%, 조명 n%, 등등)
    - ESS 설정
      - ESS 용량 (kWh)
      - 최대 충/방전 속도 (kW)
      - SOC 범위 (예: 20~95%)
      - 효율 (%) 설정
      - 운영 전략 선택: 요금최소화 / 자가소비 극대화 등
    - 전기요금 설정
      - 계약 용량 (kW)
      - 요금제 유형 선택
        - 단일요금제
        - 시간대별 요금제 (예: 경부하/중간부하/최대부하)
      - 요금표 직접 입력 또는 자동 연동
    - 실행 버튼
      - [ 시뮬레이션 실행 ]
      - [ 시나리오 저장하기 ] / [ 불러오기 ]
  - 결과 페이지(탭2)
    - 기존 조건 실시간 변경??
    - 예측 요약 정보 박스
      - 예측 발전량 합계: XXX kWh
      - 예측 소비량 합계: XXX kWh
      - 전기 요금 예측: XXX 원
      - 자가소비 비율: XX%
    - 예측 그래프
      - 발전량
        - 시간별 발전량 예측
          - 총 발전량 / 태양광 / 풍력 / 조력 각각 색상 구분
      - 소비량
        - 부식방지, 빙결방지, 조명, 기타 등
      - ESS 운영
        - SOC(충전율) 변화
        - 충전/방전 구간 표시
        - 효율 시각화
      - 마우스 오버 시 수치 표시
    - 충방전 계획
      - 운영 전략 선택
        - 요금 절감 중심
        - 자가소비 최대
        - 온실가스 배출 최소화
        - ESS 수명 보존 우선
      - 시간대 별 충전, 방전 지시표
        - 시간,발전량,소비량,잉여전력,충전,방전,한전구매
    - 요금 계산 내역
      - 총 요금
      - ESS로 절감된 요금
      - 시간대별 구매 전력량/요금
    - 결과 다운로드
      - [ PDF 다운로드 ] 버튼
      - [ Excel 다운로드 ] 버튼


○ 시뮬레이션 결과에 따른 충방전 제어 알고리즘 개발
  - ESS 충방전 스케줄 생성
    - 시간별 에너지 생산 및 소비 예측 정보 활용
    - 시간별 필요 에너지 및 생산 에너지를 활용한 충전, 방전 지시
  - 최소비용/최대효율/온실가스 감축 등 계산하여 보여줌

○ 외부 시스템 연동
  - 통합관제시스템과 데이터 연동
  - 기상청 등 공공 데이터 서비스와 연동
